# Cold-Start OS Skill

A Claude Code skill for morning cold-start protocol — AI Chief of Staff activation.

## What It Does

Recovers context from previous sessions, processes signals, generates active questions, designs experiments, and produces artifacts. Optimizes for momentum over novelty.

## Core Model

```
Theme → Thread → Signal → Question → Experiment → Artifact → Momentum
```
